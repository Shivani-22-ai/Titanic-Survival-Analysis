# Titanic Survival Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to identify factors that influenced passenger survival.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Data Exploration
- Missing Value Analysis
- Data Cleaning
- Feature Engineering
- Survival Analysis
- Data Visualization

## Key Findings
- Female passengers had a higher survival rate than males.
- First-class passengers survived more frequently than passengers in lower classes.
- Passengers travelling with family are more likely to survive.
- Children and middle age passengers are more likely to survive.
